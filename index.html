About this application

Piskel Clone is an online, in-broswer application for easily creating static pixel art images and animated pixel art gifs. The program is easy to use and offers many features, such as the ability to live preview your animations and the ability to save images and gifs! This application features:

  • Small animation preview window in the top right corner
  • Ability to add/delete, reorder, duplicate a frame
  • Export the final result in gif, png format
  • Such tools as a pen, stroke, paint bucket, rectangle and others...
